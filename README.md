🏡 Vacation Rental Platform

A full-stack web application that allows users to discover, create, and manage property listings seamlessly, inspired by real-world platforms like Airbnb.

🌐 Live Demo:
https://vacation-rental-platform-chi.vercel.app/listings

🚀 Features
🔹 Create, edit, and delete property listings (CRUD)
🔹 User authentication & authorization
🔹 Interactive maps for location-based exploration
🔹 Add and manage reviews for listings
🔹 Responsive design for smooth user experience across devices
🛠️ Tech Stack

Frontend:

EJS
HTML5, CSS3, JavaScript

Backend:

Node.js
Express.js

Database:

MongoDB

Other Tools & Services:

Mapbox (Maps & Location)
Git & GitHub
